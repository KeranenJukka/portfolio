Sirkussirkus Online Store

This is my biggest project. It's an online store made with React and Redux. The whole website is based on a single JSON file and when you update the file, products and pictures render to the webpage automatically. Since this was a learning project, I wanted to make this without additional libraries. This is also being updated from time to time. The website is designed by Kaj-Mikael Schütt (owner of the online store) and me.
